Question-1: What are the different ways to select an element in the DOM?
There are several ways to select elements in the DOM (Document Object Model) using JavaScript. Here are the main ones:
1.By ID : document.getElementById("id")
2.By Class : document.getElementsByClassName("class")
3.By Tag : document.getElementsByTagName("tag")
4.By Name : document.getElementsByName("name")
5.CSS Selector (single) : document.querySelector("selector")
6.CSS Selector (all) : document.querySelectorAll("selector")
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML, innerText, and textContent are all properties used in JavaScript to get or set content inside HTML elements, but they behave differently. Here's a breakdown of the differences:
1. innerHTML: Gets/sets HTML code (includes tags).
2. innerText: Gets/sets visible text (ignores hidden content, slower).
3. textContent: Gets/sets all text (including hidden, faster than innerText).
Question-3: What is event delegation in the DOM?
Event delegation is a technique where a parent element handles events from its child elements using a single event listener, often using event.target to identify the clicked child. It's efficient and works for dynamically added elements.
